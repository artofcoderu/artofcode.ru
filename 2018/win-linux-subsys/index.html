Добавление подсистемы Linux в Windows

ArtOfCode
17.04.2018
linux, ps, windows

В Windows 10 есть возможность включить и добавить подсистему Linux через консоль PowerShell. Это будет полезно если установку нельзя осуществить через Windows Store или для автоматизации процесса.

Ниже описан данный процесс по шагам.

  1. Нужно запустить консоль PwerShell от администратора и выполнить команду (если попросит перезагрузить, то позволить выполнить перезагрузку).

  2. В той же консоли PowerShell запущенной от администратора, выполнить команду (вместо wsl-ubuntu-1604 можно указать wsl-opensuse-42 для добавления OpenSUSE)

  3. Выполнить распаковку.

  4. Перейти в папку с распакованным дистрибутивом.

  5. Запустить исполняемый файл ubuntu.exe или fedora.exe

  6. В процессе установки попросит задать имя пользователя и пароль

  7. После установки запустить bash.exe и выполнить обновление дистрибутива двумя командами.
